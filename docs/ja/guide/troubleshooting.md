# トラブルシュート

## Godot を CLI から起動できない

- `Godot_v4.6.1-stable_win64.exe` の実体パスを指定しているか確認してください
- リポジトリのルートで実行するか、`--path <repo-root>` を付けてください
- `PATH` に入っていなくてもフルパス指定なら問題ありません

## 画面が切れて見える

- このプロジェクトは `1360x768` のビューポートを前提にしています
- `project.godot` の stretch 設定を変えていたら戻してください

## キャラクターの大きさが変に見える

- 素材ごとに元フレームサイズが揃っていません
- 計測ツールを回して `scripts/main.gd` のメトリクス辞書を更新してください
- 更新後は smoke test を再実行してください

## docs のビルドに失敗する

- まず `docs/` で `npm ci` を実行してください
- Node.js のバージョンが古すぎないか確認してください
- 画像やアイコンは `docs/public/` 配下に置くと安定します

## GitHub Actions の smoke test が落ちる

- ワークフローは公式 Windows ビルドの Godot を毎回ダウンロードします
- ダウンロード先やバージョンを変えたときは `.github/workflows/ci.yml` も合わせて更新してください
- 先に同じコマンドをローカルで通すと切り分けがしやすいです
