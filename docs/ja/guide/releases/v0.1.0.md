# v0.1.0 リリースノート

![Penguin Promenade v0.1.0 release header](/images/releases/v0.1.0-header.png)

`v0.1.0` は Penguin Promenade の初回パブリックリリースです。このページでは、このタグに含まれる出荷履歴全体をまとめています。

## ハイライト

- 最初のプレイアブル版として、夕景のアベニューを横移動で散歩し、カメラ追従と HUD の案内に従ってランドマークを巡れる構成を収録しました。
- プレイヤーは専用の走行 WebP を使い、待機、見回し、リアクション用のフレーム群を切り替えることで、ランドマークごとに異なる反応を見せます。
- 残りの animated WebP は街の住人として再利用され、位置、向き、速度、スケールに揺らぎを持つ 6 体の NPC が通りに動きを加えます。
- スプライト倍率はフレーム全体の大きさではなく可視 bounds を基準に調整しており、再計測用に `uv` 管理の Python ツールも含めています。
- 英語と日本語の README、VitePress ドキュメント、ゲームプレイスクリーンショットもこのリリースの一部です。
- GitHub Actions では docs build、GitHub Pages 公開、GitHub Release 公開後の Windows 配布物生成を自動化しています。

## 収録内容

### ゲームプレイ

- `scenes/main.tscn` と `scripts/main.gd` が、散歩ループ、3 つのランドマーク、街のペンギンたちを含むプレイ本体を構成します。
- `assets/player_frames/run` がプレイヤーの歩行アニメーション、`assets/tiles_frames` が待機、リアクション、NPC 用アニメーションを担当します。

### ツールと自動化

- `tests/smoke_test.gd` では、アニメーション設定、移動、向き、ランドマーク進行、NPC サイズ制約を検証できます。
- `tools/measure_animation_metrics.py` は `uv run python ...` で可視 bounds を計測し、アニメーション倍率調整を再現可能にします。
- `.github/workflows/ci.yml`、`.github/workflows/pages.yml`、`.github/workflows/release.yml` が docs 検証、Pages 公開、Windows 配布物生成を担当します。

### ドキュメントとリリース用アセット

- `docs/public/images/releases/v0.1.0-header.png` を今回のリリースヘッダーとして使用します。
- `docs/public/images/preview-walk.png`、`docs/public/images/preview-interact.png`、`docs/public/images/preview-npc.png` がゲームプレイの様子を示します。
