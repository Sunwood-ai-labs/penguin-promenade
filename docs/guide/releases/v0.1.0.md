# v0.1.0 Release Notes

![Penguin Promenade v0.1.0 release header](/images/releases/v0.1.0-header.png)

`v0.1.0` is the first public release of Penguin Promenade. These notes cover the full shipped history included in the tag.

## Highlights

- The initial playable build ships a compact side-scrolling promenade with a looping sunset avenue, camera follow, HUD guidance, and landmark interactions instead of combat or stage clears.
- The player animation stack uses one dedicated running WebP plus separate idle, look, and wave pose frame sets so each landmark can trigger a different response.
- Six ambient town penguins reuse the remaining animated WebP clips as smaller randomized NPCs with controlled position, facing, speed, and scale variation.
- Sprite scale alignment is tuned from measured visible bounds rather than raw frame size, and the repository includes a `uv`-managed Python tool to recompute those metrics.
- The public docs surface is part of the release: English and Japanese README files, VitePress pages, and gameplay screenshots are all included in the repository.
- GitHub Actions now validate the docs build, publish GitHub Pages, and package the Windows release bundle when the GitHub release is published.

## Shipped Areas

### Gameplay

- `scenes/main.tscn` and `scripts/main.gd` define the playable promenade, district flow, three landmarks, and the animated town cast.
- `assets/player_frames/run` provides the player walk cycle, while `assets/tiles_frames` covers the idle, interaction, and NPC animation roles.

### Tooling And Automation

- `tests/smoke_test.gd` covers animation setup, movement, facing, landmark progression, and NPC sizing constraints.
- `tools/measure_animation_metrics.py` measures visible sprite bounds with `uv run python ...` so animation scaling stays reproducible.
- `.github/workflows/ci.yml`, `.github/workflows/pages.yml`, and `.github/workflows/release.yml` cover docs validation, Pages deployment, and Windows release packaging.

### Docs And Release Assets

- `docs/public/images/releases/v0.1.0-header.png` is the release header image used for this launch.
- `docs/public/images/preview-walk.png`, `docs/public/images/preview-interact.png`, and `docs/public/images/preview-npc.png` show the shipped promenade loop, landmark interaction, and town NPCs.

The release prep was verified with the checks below:

- `uv run python tools\measure_animation_metrics.py assets\player_frames\run assets\tiles_frames\tile_09`
